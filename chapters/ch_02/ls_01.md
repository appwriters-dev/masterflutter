## Projects

1. Gender Enum
Write a dart program to create an enum class for gender [male, female, others] and print all values.

2. Calculator
Define a calculator class that has methods for addition, subtraction, multiplication, division and has properties like the result (that stores the result of the current operation)

3. Private Camera
Write a dart program to create a class Camera with private properties [id, brand, color, prize]. Create a getter and setter to get and set values. Also, create 3 objects of it and print all details.
